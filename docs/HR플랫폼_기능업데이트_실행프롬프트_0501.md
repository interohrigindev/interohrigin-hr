## 2. 채용 / 지원자 관리

### 2.1 지원자 상세 + 공유 링크

```text
채용 / 지원자 관리 기능을 보강해줘.

[요구사항]
- 지원자 상세 페이지에 면접 일시, 면접관, 면접 결과, 진행 단계, 단계별 코멘트를 추가
- 대표님 등 외부 공유용 읽기 전용 링크를 생성할 수 있게 구현

[검증]

- 로그인 없이 공유 링크 열람 가능
- 새 지원자 접수 알림 메일 수신 확인
- 최신 지원자 접수 기준 정렬 확인
```

## 3. 전자결재 / 업무보고

### 3.1 웹/모바일 분리 + 지출결의서 개편

```text
전자결재와 업무보고 작성 UX를 웹/모바일에 맞게 분리하고 지출결의서를 전면 개편해줘.

[요구사항]
- 모바일은 기존 모달 방식 유지
- 웹은 전체 페이지 작성 방식으로 전환
- 연장근무 신청 시간은 30분 단위 선택으로 변경
- 지출결의서에 입금 계좌 정보 추가
- 상세 내역 입력칸 확대
- Rich Text Editor 적용
- 본문 이미지 업로드 및 증빙 첨부파일 추가
- 첨부파일 미리보기 제공

[검증]
- 웹에서 결재 문서가 전체 화면으로 열림
- 모바일은 기존 플로우 유지
- RTE 본문/첨부/미리보기 정상 동작
- 30분 단위 시간 선택 확인
```
